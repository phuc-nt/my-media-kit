# 2026-04-10

Project v2 bootstrap day. Pivoted from Swift/SPM scaffold → Tauri v2 (Rust + HTML/JS) for cross-platform reach.

## Shipped

- Cargo workspace `src-tauri/` + 7 crates: `creator-core`, `silence-kit`, `media-kit`, `transcription-kit`, `ai-kit`, `content-kit`, `nle-kit`.
- `creator-core`: domain models (TimeRangeMs, CutRegion/Reason, SilenceRegion, TranscriptionSegment/WordTimestamp, FillerDetection, AiPromptDetection, NleExportTarget, AiProviderType, AbortFlag, CreatorError).
- `silence-kit`: DSP pipeline — frame RMS, P15/P75 auto-threshold, spike removal, region building, padding, invert-to-keeps.
- `media-kit`: ffmpeg/ffprobe resolver (env + PATH + win `.exe`), command builders (extract PCM, probe duration, cut+concat filter_complex), WAV parser, async runners.
- `ai-kit`: `Provider` trait + Claude (tool_use), OpenAI (json_schema strict), Gemini (responseSchema), Ollama (format) providers; `ProviderRegistry` with platform-gated local providers; Keyring + in-memory SecretStore.
- `nle-kit`: FCPXML 1.11 builder (rational time, fps buckets), xmeml v5 builder (timebase + ntsc).
- `transcription-kit`: Transcriber trait + NullTranscriber + WhisperModelId catalog (12 models + HF URLs + sizes). whisper-rs backend deferred (needs cmake).
- `content-kit`: transcript batching, filler/ai-prompt detectors, Summary (brief/key-points/action-items + consolidation), Chapter extractor.
- Tauri commands wired: `app_version`, `platform_info`, `media_probe`, `media_extract_pcm`, `detect_silence_in_file`, `ai_provider_status`, `ai_has/set/delete_api_key`, `ai_ping`, `nle_export`.
- Frontend skeleton: `src/index.html` sidebar + 5 feature tabs, `styles/main.css` dark theme, JS modules for sidebar/header/autocut/settings.
- `docs/architecture-decisions.md`: 11 ADRs covering the pivot.
