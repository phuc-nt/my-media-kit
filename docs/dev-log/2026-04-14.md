# 2026-04-14

Acceptance-test session on real full-length videos. Five bugs discovered and fixed end-to-end; backend pipeline test suite added covering all three.

## Fixed

- **NaN JSON parse** (`transcription-kit`): `mlx_whisper` Python JSON emits bare `NaN`/`Infinity`/`-Infinity` for silent segments with no logprob. `serde_json` rejects these. `sanitize_python_json()` now byte-walks the output, tracks string context, and replaces bare NaN/Infinity tokens with `null`. [crates/transcription-kit/src/mlx_whisper.rs](../../src-tauri/crates/transcription-kit/src/mlx_whisper.rs)

- **Mojibake Vietnamese** (`transcription-kit`): same `sanitize_python_json()` was doing `out.push(b as char)` — Latin-1 cast. Byte `0xE1` (start of `ạ`) became U+00E1, re-encoded as 2 UTF-8 bytes, exploding multi-byte sequences. Fixed: collect into `Vec<u8>`, call `String::from_utf8()` at end. Only the `null` substitution writes ASCII — UTF-8 invariant preserved throughout. [crates/transcription-kit/src/mlx_whisper.rs](../../src-tauri/crates/transcription-kit/src/mlx_whisper.rs)

- **Whisper hallucination loop** ("The Harvard men never ask that question." ×40+): `condition_on_previous_text=True` default causes cascade when model gets stuck on a silent stretch. Added `--condition-on-previous-text False` + `--hallucination-silence-threshold 2.0` to spawn args. [crates/transcription-kit/src/mlx_whisper.rs](../../src-tauri/crates/transcription-kit/src/mlx_whisper.rs)

- **Translate length mismatch** ("expected 19 translations, got 18"): 7B local model drops a line when batch is large. Three-part fix:
  1. Batch window reduced 45 s → 25 s to keep prompt smaller.
  2. `translate_batch_with_retry()` retries once with explicit count instruction.
  3. `align_to_originals()` pads missing tail with source text / truncates overflow — never errors on drift.
  [crates/content-kit/src/translate.rs](../../src-tauri/crates/content-kit/src/translate.rs)

- **`mlx_lm.server` model-name rejection**: newer versions (0.31.x) reject `POST /v1/chat/completions` when `model` field is not in the server's known list (tries HuggingFace fetch → 404). `MlxLmProvider.complete()` now calls `server_model()` on first use — fetches actual model id from `GET /v1/models`, caches in `OnceCell`, overrides `CompletionRequest.model` for every call. [crates/ai-kit/src/providers/mlx_lm.rs](../../src-tauri/crates/ai-kit/src/providers/mlx_lm.rs)

- **Translate JSON truncation** (EN clip): `max_tokens=2048` default was too small for translated Vietnamese output on a 25-second batch. Raised to `4096` in `translate_batch_with_retry` and its retry request. [crates/content-kit/src/translate.rs](../../src-tauri/crates/content-kit/src/translate.rs)

## Shipped

- **Streaming transcription progress**: `MlxWhisperTranscriber` spawns whisper with piped stdout, parses `[MM:SS.sss --> MM:SS.sss]` progress lines, calls `ProgressCallback` with `end_ms`. Tauri command `mlx_whisper_transcribe` emits `mlx_whisper_progress` events `{current_ms, total_ms, percent}`. Frontend progress bar + `%` value updates live. [crates/transcription-kit/src/mlx_whisper.rs](../../src-tauri/crates/transcription-kit/src/mlx_whisper.rs) · [src-tauri/src/commands/transcription.rs](../../src-tauri/src/commands/transcription.rs)

- **Save to file** (Transcribe + Translate): `save_text_file(path, content)` Tauri command writes UTF-8 to any absolute path, creates parent dirs if needed. Frontend derives sibling paths via `deriveSiblingPath()` (replaces extension). `.srt` + `.txt` save buttons added to both Transcribe and Translate views. [src-tauri/src/commands/files.rs](../../src-tauri/src/commands/files.rs) · [src/js/util.js](../../src/js/util.js)

- **Frontend helpers** in `util.js`: `msToSrtTimestamp`, `segmentsToSrt`, `segmentsToPlainText`, `deriveSiblingPath`, `showToast`. CSS: progress bar with indeterminate animation, running status badge, toast component.

- **E2E backend pipeline test suite** (`src-tauri/tests/end_to_end_content_pipeline.rs`): 3 tests on real full-length clips in `test-input/`. Each test: probe → `mlx_whisper` transcribe → assert (mojibake, runaway-loop ≤5 repeats, segment count > 0) → save `.transcript.srt` → translate (if `mlx_lm.server` up) → assert (skip-rule, length match) → save `.vi.srt`. Requires Apple Silicon; server-gated via `Option<MlxLmProvider>`.

## Changed

- Translate default model in UI changed to `mlx-community/gemma-4-E4B-it-4bit` (then reverted to Qwen2.5-7B in practice — gemma4 not yet supported by mlx-lm 0.31.2).
- `mlx_lm.server` model-name handling: provider now auto-detects loaded model instead of trusting caller-supplied string.

## Known follow-ups

- `gemma-4-E4B-it-4bit` not loadable by mlx-lm 0.31.2 (`ModuleNotFoundError: No module named 'mlx_lm.models.gemma4'`). Watch for mlx-lm update.
- Progress bar shows elapsed whisper position but total duration probe runs `ffprobe` separately from whisper — minor overhead.
- `test-input/*.vi.srt` output not reviewed for translation quality; only structure/length asserted.
